# /product-lookup -- Cisco Product Lookup

Look up Cisco UCS product information by PID (product ID) or code name. Returns a table with code name, PID, category, and description.

## Invocation

```
/product-lookup Alicante
/product-lookup UCSC-M-v6Q50G
/product-lookup alameda
```

## Workflow

### Step 1: Accept Input

Require a PID or code name from the user. If none provided, ask: "Enter a PID or code name to look up."

### Step 2: Search Embedded Data

The product data is embedded below in the **Product Data** section. Parse the JSON array and search for matches:

- **Case-insensitive partial match** on both `code_name` and `pid`
- Trim and lowercase the user's input before matching
- Match if input appears anywhere in `code_name` (e.g., "alameda" matches "Alameda 1", "Alameda 2")
- Match if input appears anywhere in `pid` (e.g., "UCSC-M" matches "UCSC-M-v6Q50G"; "ECMC-10G" matches rows with "UCSXE-ECMC-10G, UCSXE-ECMC-1G")
- For multi-PID rows (comma-separated in `pid`), match if input appears in any part

### Step 3: Output Table

If matches found, render a markdown table:

| Code Name | PID | Category | Description |
|-----------|-----|----------|-------------|
| ... | ... | ... | ... |

If no matches: "No products found for '[input]'. Try a different PID or code name."

### Step 4: Offer Follow-ups (optional)

- "Want me to look up another product?"
- "Need more detail on any of these?"

## Notes

- Multiple matches (e.g., "Alameda") return all matching rows
- Empty PID: product is still searchable by code name
- To update data: replace the JSON block below with new export from the Excel source
